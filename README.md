# 🤖 Rivvy — Discord Bot

> *Made by [Gab8bit](https://rivvybot.gab8bit.net/) with ❤️*

Rivvy è un bot Discord tuttofare per il tuo server. Minigiochi, media, meme, statistiche e molto altro... tutto in un unico bot.

---

## ✨ Comandi

| Comando | Descrizione |
|---|---|
| 🖼️ `/media` | Invia una foto, un video o un audio da un archivio interno di file |
| 🎵 `/musica` | Riproduce in chat vocale un file audio caricato da te |
| 📊 `/stats` | Mostra le tue statistiche nel server. Accumula **pesci** stando in VC o mandando messaggi e spendili nei minigiochi! |
| 🏆 `/classifica` | Mostra la classifica globale del server |
| 🃏 `/scommetti` | Scommetti i tuoi pesci ai minigiochi |
| 😂 `/meme` | Invia un meme da un subreddit casuale o scelto da te |
| 🤫 `/segreto` | Smaschera i segreti più intimi dei tuoi amici |
| 🛞 `/ruota` | Dai delle opzioni e lascia che la ruota decida per te |
| 🗣️ `/ripeti` | Fai ripetere un messaggio a Rivvy |
| 😮 `/troll` | Trolla qualcuno con comandi fake di kick e ban |
| 🔐 `/password` | Genera una password sup3rs1kura!!! |
| 📃 `/incorpora` | Invia un messaggio incorporato (embed) — solo i bot potevano farlo, ora puoi anche tu! |

---

## 💰 Sistema Pesci

I **pesci** 🐟 sono la valuta del server. Si accumulano:
- 📨 Mandando messaggi in chat
- 🎙️ Stando in chat vocale

E si spendono nei minigiochi di `/scommetti` (Rosso o Blu, Blackjack, ecc.).

---

## 📊 Dashboard Web

Il bot è affiancato da una web dashboard disponibile su [rivvybot.gab8bit.net](https://rivvybot.gab8bit.net/) con login via Discord OAuth2, classifica e statistiche.

---

## 🏗️ Stack Tecnico

- **Runtime:** Node.js
- **Framework:** [discord.js](https://discord.js.org/)
- **Database:** MySQL (con `mysql2` e connection pooling)
- **Process Manager:** PM2
- **Web Dashboard:** PHP + Apache2
